# MVP Roadmap

This roadmap keeps the first version small enough to finish quickly.

## Phase 1: Foundation

- Project setup
- Landing page
- Module cards
- Basic styling
- Product notes

## Phase 2: Core user flow

- Module navigation
- Chat-style input
- Quick prompts
- Simple loading and empty states

## Phase 3: Useful modules

- Daily Life Assistant flow
- Job and Freelance Coach flow
- Meal and Budget Planner flow

## Phase 4: AI connection

- Server-side chat endpoint
- Basic prompt routing by module
- Error handling
- Safe environment variable setup

## Phase 5: Beta polish

- Save recent chats
- Mobile layout polish
- README setup guide
- Deployment checklist
